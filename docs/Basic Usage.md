# How to launch

After installation with `poetry install`, or `pip install ./pkg` if you have a binary distribution;
you can launch Fouine with `fouine`.

Multiple options can be provided:
"""

    -i / --input    -> Path where EWF shall be found.  
    -o / --output   -> 
    -c / --config
    -v / --verbose
    -l / --logging

"""
